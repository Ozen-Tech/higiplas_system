# Ícones do PWA

Esta pasta deve conter os ícones do Progressive Web App (PWA).

## Ícones Necessários

- `icon-192x192.png` - Ícone 192x192 pixels
- `icon-512x512.png` - Ícone 512x512 pixels

## Como Gerar os Ícones

1. Use uma imagem base (por exemplo, o logo da empresa)
2. Redimensione para os tamanhos necessários
3. Salve os arquivos nesta pasta com os nomes corretos

## Ferramentas Recomendadas

- [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- Ferramentas de design como Figma, Canva, etc.

## Nota

Os ícones são necessários para que o PWA funcione corretamente em dispositivos móveis e para a instalação do aplicativo.
